> 组件均位于 `src/components/`，通过 `unplugin-vue-components` 自动按需注册，**无需手动 import 组件名**（工具函数仍需手动引入）。

## 页面布局

### PageShell

**路径：** `src/components/PageShell/index.vue`

全屏内容页容器，适用于表单页、详情页等带底部操作栏的页面。

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `actionBar` | `boolean` | `true` | 是否为底部固定操作栏预留空间 |

```vue
<PageShell>
  <CardSection title="基本信息">...</CardSection>
  <StickyActionBar>...</StickyActionBar>
</PageShell>
```

---

### CardSection

**路径：** `src/components/CardSection/index.vue`

卡片分组：可选标题 + `BaseCard` + `van-cell-group`。

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | `""` | 分组标题 |
| `variant` | `'default' \| 'form'` | `'default'` | `form` 时统一 label / 必填星号 / 内容区对齐 |
| `labelWidth` | `string` | `'6.2em'` | 与 `van-form` 的 `label-width` 保持一致 |

- **`variant="default"`**：详情展示，cell 值默认右对齐
- **`variant="form"`**：表单编辑，label 固定宽度、内容左对齐、标题竖线与 field 对齐

```vue
<van-form label-width="6.2em">
  <CardSection title="基本信息" variant="form">
    <van-field label="姓名" ... />
    <AreaPicker v-model="region" field-label="所在地区" required />
  </CardSection>
</van-form>
```

**路径：** `src/components/BaseCard/index.vue`

基础卡片容器（纯样式）。

| Prop | 类型 | 默认值 | 说明 |

| `shadow` | `'none' \| 'sm' \| 'md'` | `'sm'` | 阴影 |
| `border` | `boolean` | `false` | 是否显示边框 |
| `radius` | `number \| string` | `12` | 圆角 |
| `padding` | `number \| string` | `'12px 16px'` | 内边距，传 `0` 可去掉 |

### StickyActionBar

**路径：** `src/components/StickyActionBar/index.vue`

底部固定操作栏，含 safe-area 适配。

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `fixed` | `boolean` | `true` | 是否 fixed 贴底 |
| `placeholder` | `boolean` | `true` | 是否渲染占位块 |
| `placeholderHeight` | `string` | `""` | 自定义占位高度 |
| `border` | `boolean` | `true` | 顶部分割线 |
| `blur` | `boolean` | `true` | 背景模糊 |
| `padding` | `string` | `'12px 16px'` | 内边距 |
| `zIndex` | `number \| string` | `100` | 层级 |

默认 slot 内按钮会自动 `flex: 1` 均分宽度。

## 表单与选择器

以下选择器均支持 `#trigger` 插槽自定义触发区域，以及 `fieldLabel` / `placeholder` / `disabled` 等通用属性。

### AreaPicker

**路径：** `src/components/AreaPicker/index.vue`

省市区选择（`van-area` + 弹出层）。

| Prop | 说明 |
| --- | --- |
| `v-model` | 选中地区对象，未选为 `null` |
| `columnsNum` | `1` 省 / `2` 省市 / `3` 省市区（默认） |
| `areaList` | 自定义地区数据，默认 `@vant/area-data` |
| `required` | 是否显示必填星号 |

**v-model 结构：**

```js
{
  code: "330106",
  label: "浙江省 / 杭州市 / 西湖区",
  province: "浙江省",
  city: "杭州市",
  county: "西湖区",
  codes: ["330000", "330100", "330106"]
}
```

## 列表与分页

### ScrollRefreshList

**路径：** `src/components/ScrollRefreshList/index.vue`

下拉刷新 + 上拉加载列表容器，配合 `usePagedList` 使用。

| Prop | 说明 |
| --- | --- |
| `mode` | `'box'` 局部滚动（需传 `height`）/ `'page'` 整页滚动 |
| `v-model:refreshing` | 下拉刷新状态 |
| `v-model:loading` | 上拉加载状态 |
| `v-model:error` | 加载失败状态 |
| `finished` | 是否已全部加载 |
| `empty` | 是否展示空态 |
| `showBackTop` | 是否显示回到顶部 |

---

### usePagedList

**路径：** `src/hooks/usePagedList.js`

分页列表逻辑 Hook，详见组件内 JSDoc 与 [列表示例页](#示例页面)。

```js
const { list, refreshing, loading, finished, error, isEmpty, onRefresh, onLoad, reload, reset } =
  usePagedList({
    fetcher: ({ page, pageSize }) => getListApi({ page, pageSize }),
    getList: res => res.list,
    getTotal: res => res.total,
    pageSize: 10
  });
```

**常用方法：**

| 方法 | 说明 |
| --- | --- |
| `onRefresh` | 下拉刷新，重置到第 1 页 |
| `onLoad` | 上拉加载下一页 |
| `reload` | 手动刷新（筛选条件变化后） |
| `reset` | 清空列表和分页状态 |

## 上传与预览

### ImageUpload

**路径：** `src/components/ImageUpload/index.vue`

图片上传组件。

| Prop | 说明 |
| --- | --- |
| `v-model` | 已上传图片 URL 数组 |
| `uploadFn` | 自定义上传函数 `(file) => Promise<string>`，优先级最高 |
| `uploadUrl` | 内置上传地址，默认读 `VITE_UPLOAD_URL`；为空则仅本地预览 |
| `maxCount` | 最大数量，默认 `9` |
| `maxSize` | 单文件大小限制（字节） |
| `maxConcurrent` | 上传并发数，默认 `1` |

### FileUpload

**路径：** `src/components/FileUpload/index.vue`

通用文件上传组件，API 与 `ImageUpload` 类似，支持 `accept`、`previewImage`、`deletable` 等。

---

### FilePreview

**路径：** `src/components/FilePreview/index.vue`

附件列表预览与下载。

| Prop | 说明 |
| --- | --- |
| `files` | 文件数组或 URL 字符串数组 |
| `showDownload` | 是否显示下载按钮 |
| `clickToPreview` | 点击整行时尝试预览（仅可预览类型） |

**行为说明：**

- 可预览类型（图片、视频、音频、PDF、文本等）：显示「预览 + 下载」
- 不可预览类型：**仅显示下载**（无「打开」按钮）

**事件：** `download` / `preview` / `preview-image`

## 工具函数 utils

### validate.js

**路径：** `src/utils/validate.js`

表单校验工具，可直接用于 Vant Field 的 `:rules`。

**正则常量：**

| 常量 | 说明 |
| --- | --- |
| `MOBILE_REG` | 中国大陆手机号 |
| `EMAIL_REG` | 邮箱 |
| `ID_CARD_REG` | 18 位身份证 |
| `SMS_CODE_REG` | 6 位短信验证码 |
| `CHINESE_NAME_REG` | 中文姓名（2–20 位） |

**校验函数：**

| 函数 | 说明 |
| --- | --- |
| `isMobile(val)` | 手机号 |
| `isEmail(val)` | 邮箱 |
| `isIdCard(val)` | 身份证（含校验码） |
| `isSmsCode(val)` | 短信验证码 |
| `isChineseName(val)` | 中文姓名 |
| `isExternal(path)` | 是否外链（`https:` / `mailto:` / `tel:`） |

**Vant Field 规则生成器：**

| 函数 | 说明 |
| --- | --- |
| `ruleRequired(message?)` | 必填 |
| `ruleValidator(validator, message)` | 自定义校验 |
| `ruleMobile(message?)` | 手机号 |
| `ruleEmail(message?)` | 邮箱 |
| `ruleIdCard(message?)` | 身份证 |
| `ruleSmsCode(message?)` | 短信验证码 |
| `ruleChineseName(message?)` | 中文姓名 |

```vue
<van-field
  v-model="form.mobile"
  label="手机号"
  :rules="[ruleRequired('请输入手机号'), ruleMobile()]"
/>
```

---

### format-time.js

**路径：** `src/utils/format-time.js`

时间格式化与 Picker 值转换。

| 函数 | 说明 |
| --- | --- |
| `formatTime(date, pattern?)` | 按模板或预设格式化 |
| `getCurrentTime(format?)` | 获取当前时间字符串 |
| `getCurrentTimestamp()` | 当前时间戳（毫秒） |
| `isValidDate(value)` | 是否有效日期 |
| `toDatePickerValues(value)` | 转 DatePicker 数组 `['2026','06','18']` |
| `toTimePickerValues(value, withSecond?)` | 转 TimePicker 数组 |
| `fromDatePickerValues(values)` | DatePicker 数组 → `YYYY-MM-DD` |
| `fromTimePickerValues(values, withSecond?)` | TimePicker 数组 → `HH:mm` |
| `parseDateTimeString(value)` | 解析 `YYYY-MM-DD HH:mm(:ss)` |

**预设格式：** `年月日` / `年月日时分` / `年月日时分秒` / `时分` / `date` / `datetime` / `time` 等。

### http/index.js

**路径：** `src/utils/http/index.js`

Axios 封装实例，导出 `http`。

- 自动注入 `Authorization: Bearer <token>`
- 请求/响应拦截 + NProgress 进度条
- 业务码 `200` 成功，返回 `result` 字段
- `401` 或 HTTP 401 自动清登录态并跳转登录页

```js
import { http } from "@/utils/http";

http.request({ url: "/xxx", method: "get", params: { id: 1 } });
```

## 快速组合参考

### 表单页

  </van-form>
  <StickyActionBar>

```

  mode="page"
  v-model:refreshing="refreshing"

  v-model:error="error"
  :finished="finished"
  :empty="isEmpty"
  @refresh="onRefresh"
  @load="onLoad"

  <van-cell v-for="item in list" :key="item.id" :title="item.title" />

```
